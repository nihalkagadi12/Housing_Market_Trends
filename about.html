About the Project

The project "Visualizing Housing Market Trends: An Analysis of Sale Prices and Features using Tableau" focuses on analyzing housing market data to understand patterns and trends in property sales.

Using Tableau, interactive visualizations and dashboards are created to explore how different factors such as house size, number of bedrooms, neighborhood, and overall property quality influence housing prices.

The main goal of this project is to present housing data in a clear and interactive way so that users can easily understand market behavior and price variations.

This analysis helps home buyers, real estate agents, and investors make better decisions by identifying key factors that affect housing prices.

The project demonstrates how data visualization tools like Tableau can transform raw data into meaningful insights through charts, dashboards, and data stories.
